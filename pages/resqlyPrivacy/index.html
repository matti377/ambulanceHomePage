Privacy Policy for ResQly

ResQly does not collect, store, or share personal data.

Data Collection

We do not collect any personal information such as names, emails, or account data. The app can be used without creating an account.

Permissions

ResQly may request access to certain device features:

  • Location: Used to provide location-based functionality such as nearby information. Location data is not stored or transmitted.
  • Camera: Used only for scanning QR codes for authentication or configuration purposes. No images or video are stored.

Third-Party Services

Some features use external services:

  • OpenAI API (for image analysis)
  • Google Text-to-Speech API

These services may process data only when the user actively uses these features. We do not control how these services handle data. Please refer to their respective privacy policies.

Data Storage

All settings and preferences are stored locally on the user’s device. No data is transmitted to external servers by the app itself.

Security

We take reasonable measures to ensure that the app operates securely. However, no method of electronic storage is completely secure.

Changes

This privacy policy may be updated in the future. Any changes will be reflected in this document.
