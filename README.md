# Ambulance Home Page

A Progressive Web App (PWA) designed for ambulance professionals in Luxembourg, providing quick access to essential tools, information, and resources for emergency medical services.

## Features

- **ECG Interpreter**: Interactive tool for interpreting electrocardiograms with prompts and guidance.
- **CIS LUX Team**: Information about the Luxembourg ambulance service teams.
- **Hôpital de Garde**: Directory of hospitals on duty in Luxembourg.
- **Infectious Diseases**: Database of infectious disease protocols and information.
- **Info Page**: General information and resources.
- **Leveso Handouts**: Handouts and guidelines from Leveso (Luxembourg Emergency Medical Services).
- **Pedi**: Pediatric emergency protocols and data.
- **Phone Numbers**: Emergency contact numbers and directories.
- **QRF Cards**: Quick Reference Cards for rapid access to critical information.
- **SAMU Quiz**: Quiz on SAMU medications for training and review.
- **SAP Quiz**: Quiz on SAP (Système d'Alerte Précoce) with questions from CSV data.

## Technologies Used

- **HTML5**: Structure and content
- **CSS3**: Styling and responsive design
- **JavaScript**: Interactivity and functionality
- **PWA Features**: Service worker, manifest.json for offline capability and installability
- **Data Storage**: JSON files for structured data (e.g., medications, diseases)

## Installation and Setup

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/yourusername/ambulanceHomePage.git
   cd ambulanceHomePage
   ```

2. **Open in Browser**:
   - Simply open `index.html` in your web browser.
   - For the best experience, serve it via a local server (e.g., using Python: `python -m http.server`).

3. **Install as PWA** (Optional):
   - Open the app in a compatible browser (Chrome, Firefox, etc.).
   - Use the browser's install prompt or menu to add it to your home screen.

## Usage

- Navigate through the main menu to access different sections.
- Each page is self-contained with relevant information and tools.
- The app is designed for offline use once installed as a PWA.
